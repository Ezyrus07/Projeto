1 Executar SQL no Supabase

O que faz: Corrige as polticas de segurana (RLS) que esto impedindo o site de acessar os dados dos usurios.

  1. Acesse: https://app.supabase.com
  2. Entre no seu projeto
  3. Clique em "SQL Editor" no menu lateral
  4. Copie e cole o SQL abaixo:
-- Correo RLS - Tabela Usurios
alter table if exists public.usurios enable row level security;

-- Remove policies antigas
drop policy if exists "usurios_upsert_own" on public.usurios;
drop policy if exists "usurios_update_own" on public.usurios;
drop policy if exists "usurios_select_own" on public.usurios;
drop policy if exists "usurios_select_authenticated" on public.usurios;

-- SELECT (permite usurios autenticados lerem perfis)
create policy "usurios_select_authenticated"
  on public.usurios for select to authenticated using (true);

-- INSERT (permite criar prprio perfil)
create policy "usurios_upsert_own"
  on public.usurios for insert to authenticated
  with check (uid::text = auth.uid()::text);

-- UPDATE (permite atualizar prprio perfil)
create policy "usurios_update_own"
  on public.usurios for update to authenticated
  using (uid::text = auth.uid()::text)
  with check (uid::text = auth.uid()::text);
?
Como saber que funcionou:
Deve aparecer "Success. No rows returned" ou similar
